MedImage

A system for doctors and medical students to record medical procedures, featuring image capture and real-time hospital HIS integration.

  • Developed an admin management system using HTML and CSS frameworks.
  • Created a user-facing application using Native Kotlin with functions like drawing, menu, search, usage history, QR Code, and barcode scanning.
  • Implemented image-to-PDF conversion and data transmission through hospital APIs.
  • Developed APIs for retrieving and sending data between the hospital system and frontend using PHP.
  • Built backend management system in PHP and deployed on the Google Play Store [KCMH MedImage].
